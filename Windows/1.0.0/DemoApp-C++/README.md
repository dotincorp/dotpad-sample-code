# C++ Demo application for DotPadSDK Windows

## Setting
- Before build, the SDK folder should be located at /DemoApp folder like below.
- DotPadSDK library download path : https://github.com/dotincorp/dotpad-sdk-guide/tree/main/Windows/1.0.0

## Directory
```
DemoApp
├── DemoApp.sln
├── DemoApp
└── SDK
    ├── 32bit
    |   ├── DotPadSDK.dll
    |   ├── jsoncpp.dll
    |   ├── TTBEngine.dll
    |   ├── MeCab.dll
    |   ├── mecabrc
    |   ├── dot_pad_sdk.h
    |   └── dot_pad_sdk_error.h
    |
    └── 64bit
        ├── DotPadSDK.dll
        ├── jsoncpp.dll
        ├── TTBEngine.dll
        ├── MeCab.dll
        ├── mecabrc
        ├── dot_pad_sdk.h
        └── dot_pad_sdk_error.h
    |   
```
